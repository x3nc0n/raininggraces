## Fixes Applied

### 1. SWA /admin Route Protection

**File:** `staticwebapp.config.json`

Added exact `/admin` route before wildcard `/admin/*` to properly protect the bare `/admin` path. SWA route matching is first-match-wins; the wildcard does not match the root path, which would fall through to `navigationFallback` and serve the dashboard HTML to unauthenticated users.

```json
{
  "route": "/admin",
  "allowedRoles": ["authenticated"]
},
{
  "route": "/admin/*",
  "allowedRoles": ["authenticated"]
}
```

### 2. Azure Functions Rate Limiting Migration

**Files:** `api/helpers/storage.js`, `share-verify/index.js`

Migrated rate limiting from in-memory `Map` (non-persistent across Function instances) to Azure Table Storage.

**Schema:**
- Table: `ratelimits`
- PartitionKey: `ratelimit`
- RowKey: sanitized IP (replace `/`, `\`, `#`, `?` with `-`)
- Fields: `count`, `windowStart`
- Limit: 5 attempts per 60-second window

**Storage helpers exported:**
- `isRateLimited(ip)` → boolean
- `registerFailedAttempt(ip)` → void
- `resetRateLimit(ip)` → void

Table auto-created on first cold start via `ensureStorageReady()`.
